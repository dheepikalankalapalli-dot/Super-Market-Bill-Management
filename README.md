# Super-Market-Bill-Management
 

## Description
The Supermarket Bill Management System is a simple application developed to manage billing operations in a supermarket. It helps generate bills, calculate totals, and maintain product details efficiently.

## Features
- Add and manage products
- Generate customer bills
- Automatic total calculation
- Display product price and quantity
- Simple and user-friendly interface

## Technologies Used
- Python
- Basic Programming Concepts

python main.py

## Purpose of the Project
This project was developed to practice programming and understand how billing systems work in supermarkets.
